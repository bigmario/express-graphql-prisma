# GraphQL User CRUD with PrismaORM

## Run on Local
_edit ```.env.example``` file according to needs. and rename it to ```.env```_
```bash
#Installation
$ yarn
```

```bash
#Run Project
$ yarn --cwd api dev
```
## Run with Docker
```bash
$ docker-compose build
$ docker-compose up -d
```

## Access Apollo Server
```http://localhost:<port>/graphql```
